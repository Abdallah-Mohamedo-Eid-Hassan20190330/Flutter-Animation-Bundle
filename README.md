# Flutter Animation Bundle

A new Flutter project for displaying animation capabilities.

## Getting an overview: 

This project is a guide for different types of animations in flutter. 

It is divided into four main categories
- Animated Foo
- Tween Animation
- Foo Transition 
- Animated Builder
